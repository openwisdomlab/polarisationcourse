### Prompt Engineering Best Practices

**1. Be Specific**
âŒ Bad: "Make a polarization demo"
âœ… Good: "Create Malus's Law demo with I = Iâ‚€cosÂ²Î¸ formula, showing intensity vs angle plot with real-time slider control"

**2. Include Physics**

- Validation criteria

**3. Describe Visualization**

- Plot type (line, bar, 3D, polar)

- Color scheme
- Annotations

**4. Define Interactivity**

- Default value

- Class hierarchy
- Method signatures
- Expected line count
- Documentation style
